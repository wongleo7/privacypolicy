Privacy Policy

DAS Symbol Linker Chrome Extension

Last Updated: March 13, 2026

TL;DR: This extension does NOT collect, store, or transmit any personal data. Everything runs locally on your computer. No analytics, no tracking, no servers.

Overview

DAS Symbol Linker is a privacy-focused Chrome extension that helps traders send stock symbols from web pages to their DAS Trader desktop application. This privacy policy explains what data the extension accesses and how it's used.

Data Collection

We do NOT collect any data. Specifically:

  • ❌ No personal information
  • ❌ No browsing history
  • ❌ No stock symbols or trading data
  • ❌ No analytics or usage tracking
  • ❌ No cookies or identifiers
  • ❌ No data sent to external servers

How the Extension Works

1. Local Storage (chrome.storage)

What's stored: User preferences only

  • DAS Trader window title (e.g., "CenterPoint Pro")
  • Keyboard hotkey settings

Where it's stored: Locally in Chrome's storage (synced across your Chrome installations if you're signed in)

Who can access it: Only you, on your devices

2. Native Messaging (chrome.runtime.sendNativeMessage)

What's transmitted: Stock symbols (e.g., "AAPL", "TSLA")

Where it's sent: Only to a local application running on YOUR computer (com.shortscanner.das_linker)

Network transmission: None - all communication stays on your local machine

3. Content Scripts

Active on: https://stock-news-feed.vercel.app only

What it does:

  • Detects when you click stock symbols on the scanner page
  • Listens for messages from the web app
  • Shows local notifications when symbols are sent

What it does NOT do: Read, collect, or transmit any page content or personal data

4. Context Menus

What it does: Allows you to right-click selected text and send it to DAS

Data access: Only reads the text you explicitly select and right-click on

Data retention: None - text is immediately processed and discarded

Permissions Explained

Permission Why We Need It Privacy Impact
nativeMessaging Send symbols to DAS Trader desktop app ✅ Local only, no network
contextMenus Right-click to send symbols ✅ Only selected text, user-initiated
storage Save user settings ✅ Local only, user preferences
host_permissions Detect symbol clicks on scanner ✅ Single domain, no data extraction
